KEY TESTING PATTERNS:
────────────────────

1. BASIC TEST:
   when(mock.method()).thenReturn(value);
   Result result = service.method();
   assertThat(result).isNotNull();

2. ERROR TEST:
   assertThatThrownBy(() -> service.method())
       .isInstanceOf(Exception.class);

3. PAGINATION TEST:
   Page<T> page = new PageImpl<>(list, PageRequest.of(0, 10), 1);
   when(repo.findAll(any())).thenReturn(page);

DEBUGGING TIPS:
───────────────
❌ "cannot find symbol": Using getter on record
   Fix: Change getId() to id()

❌ "AssertionError: actual value: null": Mock not set up
   Fix: Add when(...).thenReturn(...) before service call

❌ "InsufficientStockException": Product stock < request
   Fix: testProduct.setStock(requestQuantity + 5)

❌ "UnnecessaryStubbing": Mock defined but not used
   Fix: Remove the when() statement or use the mock
